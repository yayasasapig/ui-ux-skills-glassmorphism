## 1. Researcher Agent

**用途：** 資訊檢索、事實查核、主題深度研究

### 強項 ✅
- 多來源網頁搜尋與內容整合
- 事實查核與數據驗證
- 競品分析、行業趨勢研究
- 學術文獻回顧

### 限制 ⚠️
- 無法訪問付費資料庫（需用戶提供）
- 生成的程式碼能力有限
- 不擅長創意寫作或翻譯

### 最佳輸入格式
```
研究主題：[具體問題]
目標受眾：[專業/大眾/學術]
需要的深度：[淺層概述/深度報告]
截止時間：[如有]
```

## 2. Translator Agent

**用途：** 多語言翻譯、文化本地化

### 強項 ✅
- 中 ↔ 英 ↔ 日 ↔ 其他語言翻譯
- 語境理解，避免直譯
- 商業/技術/文學不同風格調整
- 廣東話/繁體中文專業

### 限制 ⚠️
- 非常專業的領域術語可能需要用戶確認
- 無法翻譯圖片內文字（需配合 OCR）
- 詩詞/雙關語等創意翻譯效果有限

### 最佳輸入格式
```
原文：[文本]
來源語言：[如已知]
目標語言：[如繁中/英文]
語境：[商務/日常/學術]
風格偏好：[正式/輕鬆]
```

## 3. Tutor Agent

**用途：** 教學解釋、概念澄清、練習題生成

### 強項 ✅
- 複雜概念簡單化解釋
- 步驟式教學（數學、程式、語法等）
- 因材施教，根據理解程度調整
- 生成練習題與解答

### 限制 ⚠️
- 無法互動式監考或即時批改口語
- 深度專業課程（博士等級）能力有限
- 無法代替正式教育認證

### 最佳輸入格式
```
科目：[數學/語言/科學...]
學習者程度：[小學/初中/高中/大學]
想解決的問題：[具體問題]
教學風格偏好：[嚴謹/輕鬆/實例導向]
```

## 4. Developer Agent

**用途：** 程式開發、程式碼除錯、技術架構建議

### 強項 ✅
- 多語言程式開發（Python/JS/TypeScript/Swift/Go...）
- 程式碼除錯與優化
- API 整合與架構設計
- 開源工具與框架應用

### 限制 ⚠️
- 無法在封閉環境中執行（需用戶確認環境）
- 重大重構需要完整上下文
- 不擅長 UI/UX 設計（建議搭配專門工具）

### 最佳輸入格式
```
任務：[新功能/除錯/重構/優化]
語言/框架：[具體技術棧]
現有程式碼：[或上傳檔案]
目標環境：[雲端/本地/特定平台]
約束條件：[性能/安全性/兼容性]
```

## 5. Secretary Agent

**用途：** 行程管理、郵件處理、會議紀錄、行政事務

### 強項 ✅
- 日曆管理與行程安排
- 郵件撰寫與分類
- 會議紀錄與重點摘要
- 待辦事項追蹤

### 限制 ⚠️
- 無法直接發送真實郵件（需透過 gog skill）
- 需要明確的截止時間和優先級
- 複雜判斷決策需用戶確認

### 最佳輸入格式
```
任務類型：[行程安排/郵件/紀錄/待辦]
相關細節：[時間/對象/主題]
優先級：[高/中/低]
截止：[日期時間]
```

## 6. Image Generator Agent

**用途：** 圖像生成與編輯

### 強項 ✅
- 文字轉圖像（MiniMax Image-01）
- 漫畫風格插圖（cat-image）
- 產品展示、海報、概念圖
- 圖片編輯與變體生成

### 限制 ⚠️
- 手指、人臉等細節有時不完美
- 無法生成真實人物照片（風格限制）
- 複雜建築或文字在圖像中可能變形

### 最佳輸入格式
```
主題：[具體描述]
風格：[寫實/卡通/水彩/賽博...]
用途：[社群媒體/簡報/印刷]
尺寸：[1:1/16:9/其他]
其他要求：[如配色/構圖偏好]
```

## 7. Summarizer Agent

**用途：** 內容摘要、長文壓縮、重點提取

### 強項 ✅
- 網頁/文件快速摘要
- 會議錄音/訪談文字稿重點
- 比較多家來源的核心差異
- 結構化輸出（條列/表格）

### 限制 ⚠️
- 無法理解複雜數學公式或圖表
- 原始音訊品質差會影響準確度
- 極短內容可能無需摘要

### 最佳輸入格式
```
來源：[URL/檔案路徑/文字]
摘要長度：[一句話/三句/一段/詳細]
重點方向：[如有特別強調的面向]
輸出格式：[條列/段落/表格]
```

## 8. Generalist Agent

**用途：** 通用對話、創意寫作、簡單問題解答、多領域整合

### 強項 ✅
- 日常問題解答
- 創意寫作（故事、腳本、文案）
- 跨領域整合建議
- 快速一次性任務

### 限制 ⚠️
- 深度專業任務不如專門 Agent
- 複雜程式開發建議用 Developer Agent
- 大量內容翻譯建議用 Translator Agent

### 最佳輸入格式
```
諮詢類型：[創意寫作/建議/翻譯/分析...]
具體需求：[描述]
背景資訊：[如相關上下文]
預期輸出：[格式/長度/風格]
```
